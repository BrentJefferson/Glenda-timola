# Project Spec — Glenda Timola Real Estate

## Overview
Personal agent portfolio website for Glenda Timola, a Philippine real estate agent. Single-page application with fast load times.

## Requirements
- [x] Mobile responsive
- [x] Modern design (Modern Luxury aesthetic)
- [x] Light mode
- [x] Single-page app for quick response
- [x] Data-driven (no CMS — edit JS files in `data/`)
- [x] Caching for faster data loads
- [x] Listing cards with thumbnail + general info
- [x] Detail view with image carousel when card is clicked
- [x] Markdown-rendered listing captions from FB posts

## Sections
1. **Hero** — Full-viewport intro with name, title, tagline
2. **Services** — What she offers (buying, selling, consulting)
3. **Featured Listings** — Property cards with thumbnail + info → detail modal/page with carousel
4. **Testimonials** — Client reviews
5. **Contact** — Contact form, phone, email

## Design Direction
- **Aesthetic:** Modern Luxury — warm, premium, clean
- **Colors:** TBD (design system interview pending)
- **Typography:** Editorial — bold display font + refined body font
- **Theme:** Light mode
